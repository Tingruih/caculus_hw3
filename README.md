格式規定：
題目要貼上
答案的內容需詳細寫出運用到class's slides裡的哪些公式
work division 寫作業題目的分配
working hours record 則要詳細寫出包含作業外學習的事項（包含每一項的個別工作時數）
challenges and difficulty:
此處由每位成員報告在擔任實作者或審核者時遇到的挑戰或未解決的問題，包含未來工作

meeting record:
可以將會議記錄寫在appendix裡面，將條列式將重點寫在meeting records內的表格並引用appendix裡的會議記錄

reflection:
應包含在作業中學習到的知識點，個人和小組整題的學習，工作模式描述的反思，總結在此次作業中遇到的挑戰和問題(work hours, division)
